## 播放器结合 React 组件

### 开发

- 安装依赖 `pnpm install`

- 编译组件
  1. `cd components`
  2. `pnpm run dev` 或者生产环境 `pnpm run build`

- 运行示例启动服务 `pnpm run dev` (需要在根目录下)

- 解码资源

  复制 `dist` 下 `ezuikit_static` 、`ezuikit.js`、`ezuikit-flv` 和 `ezuikit-hls` 到 react 项目的静态资源目录下
